# Code of Conduct

Espander aims to be a welcoming open-source project.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, especially across language and experience gaps.
- Keep feedback focused on the work, not the person.
- Help make issues and pull requests clear, reproducible, and useful.

## Unacceptable Behavior

Harassment, hate speech, threats, personal attacks, spam, or deliberate disruption are not welcome in this project.

## Reporting

If you see behavior that makes the project unsafe or unwelcoming, contact the maintainer privately. Reports will be reviewed with care.
