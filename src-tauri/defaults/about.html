About Espander

Espander is an open-source visual manager and sync companion for Espanso. It helps users create snippets, organize them into categories, export or import their data, and sync YAML files through GitHub.

Built with Tauri, React, Rust, and TypeScript. No telemetry, no ads, no mandatory cloud backend.
